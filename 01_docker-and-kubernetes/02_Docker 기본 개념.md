# 2. Docker 기본 개념

## 2.1 Docker란 무엇인가?

- Containerization
  - 컨테이너화 하는 기술
- Container
  - 격리된 공간에서 프로세스를 실행시킬 수 있는 기술



## 2.2 Docker의 이점

> Build Once, Run Anywhere

- 도커에서 제공하는 패키징 방식을 통해서 하나의 어플리케이션과 관련된 모든 것들을 하나로 묶은 Docker Image 형태로 Build
- 어떤 환경이든지 상관없이 동일하게 실행할 수 있음
- Dockerize
  - Docker Image로 만드는 것
